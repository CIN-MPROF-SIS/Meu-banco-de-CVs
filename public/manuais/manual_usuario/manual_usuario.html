  1. Introdução

O SELECAD+ é um sistema Web que pode ser utilizado de qualquer lugar via internet. Esse manual tem como objetivo orientar o usuário no uso do sistema.

  1. Visão geral do sistema

O SELECAD+ foi idealizado para auxiliar a seleção de candidatos de forma “descomplicada” permitindo ao contratante selecionar vários candidatos em um único processo de forma rápida e econômica. Aumentando assim a possibilidade de encontrar candidatos com conhecimentos específicos para  a vaga anunciada.

No SELECAD+, o Contratante é a pessoa física ou jurídica que faz o cadastro para anunciar vaga de emprego e realizar o processo de seleção de candidatos para as suas vagas ofertadas. Já o Candidato é a pessoa física que faz o cadastro para concorrer a vaga de emprego. O candidato pode buscar vagas ofertadas e enviar currículos, visualizar a quantidade de acessos ao seu currículo pelos contratantes, ter acesso ao histórico das vagas inscritas e visualizar informações sobre o tempo de cadastro do contratante no SELECAD+. Ao final da seleção, o sistema gera um relatório com a classificação e pontuação dos candidatos.

Espera-se que as funcionalidades do sistema proporcione ao usuários realizarem de forma intuitiva, rápida e econômica a busca de um emprego ou a seleção de um candidato.

  1. Níveis de acesso ao sistema

O SELECAD+ possui três perfis de usuário:  Contratante (Pessoa física ou Jurídica), Candidato e Moderador.  

Moderador - é o administrador do sistema responsável por validar o cadastro do Contratante, moderar usuários, cadastrar dados para uso nos formulários do sistema e aprovar os comentários cadastrados.

Contratante - é a pessoa física ou jurídica que faz o cadastro para anunciar vaga de emprego e realizar o processo de seleção de candidatos para as suas vagas ofertadas. O cadastro é validado pelo moderador e somente depois de aprovado, o contratante poderá registrar vagas e realizar o processo de seleção dos candidatos.

Candidato - é a pessoa física que faz o cadastro para concorrer a vaga de emprego. O Candidato pode realizar a busca de vagas ofertadas e enviar currículos, ter acesso ao histórico das vagas inscritas e visualizar informações sobre o tempo de cadastro da empresa no SELECAD+. Ao participar de uma seleção, o candidato recebe notificação de cada etapa concluída (aprovado ou eliminado).

Na página inicial do SELECAD+ clique no menu Cadastro para cadastrar o seu usuário e a senha de acesso (ver imagem 1).

  1. Página do candidato

Ao logar no sistema o candidato será redirecionado para a sua página restrita (ver imagem 4).

Imagem 4 - Página inicial do candidato

Na sua página, o candidato pode realizar as seguintes atividades: Cadastrar/Atualizar perfil, Alterar sua senha, visualizar ofertas de vagas, candidatar-se a vagas e visualizar suas candidaturas.

5.1 - Cadastrar/Atualizar perfil

No primeiro acesso a página restrita, realize o preenchimento dos seus dados pessoal e profissional clicando no menu Perfil. Na página Meu Perfil preencha o formulário (ver imagem 5) com todas as informações necessárias para a seleção. As informações cadastradas podem ser alteradas a qualquer momento, basta clicar novamente em Meu Perfil e realizar novas inclusões/alterações de informações, como novas qualificações e/ou experiências profissionais, alterações de contatos, etc.

Imagem 5 - Página perfil do candidato

5.2 - Alterar sua senha

        Ao clicar no item do menu com o “login do usuário” localizado ao lado do item “Home”, será carregada a página onde será possível realizar a alteração da sua senha (ver imagem 6).

Imagem 6 - Página Alterar senha

5.3 - Visualizar ofertas de vagas

        Ao clicar no item do menu “vagas disponíveis” será exibida uma tela com todas as vagas cadastradas e disponíveis para candidaturas. Ao lado das vagas listadas terá um link “candidatar” ou “já candidatou”, caso o candidato já tenha se candidatado a essa determinada vaga (ver imagem 7).

Poderão ser realizados filtros para uma melhor seleção das vagas de acordo com a preferência do candidato.  

Imagem 7 - Página oferta de vagas

5.4 - Candidatar-se a vagas

        Na listagem de vagas (ver imagem 7), clique no item “candidatar” da vaga pretendida. Após clicar, será exibida a página com as informações sobre a vaga. Clique no botão Candidatar para confirmar a sua candidatura para essa vaga (ver imagem 8).

Imagem 8 - Página candidatar a vaga

5.5 - Minhas candidaturas

        Ao clicar no item do menu “Minhas Candidaturas”, o candidato irá visualizar todas as vagas na qual se candidatou e poderá realizar as operações de ver vaga, desistir e questionário (ver imagem 9).

        O item questionário só estará disponível se o contratante cadastrar um questionário para avaliar o candidato. Preencha-o, pois ele será visualizado pelo contratante e utilizado como filtro para a seleção.

Imagem 9 - Minhas Candidaturas

  1. Página do contratante

Ao logar no sistema o contratante será redirecionado para a sua página restrita (ver imagem 10). Nesta página é listado todas as vagas ofertadas pelo contratante. O item Mostrar exibe os dados que foram cadastrados para a vaga. O item Editar permite ao contratante alterar os dados da vaga cadastrada. O item Selecionar está descrito no tópico 6.4 deste manual e o item Questionário no tópico 6.3.

Imagem 10 - Página restrita do contratante

        O contratante pode realizar as seguintes atividades: Preencher os dados da sua pessoa física ou jurídica, Ofertar vaga de emprego, Criar formulário de questões, Selecionar Candidatos e Emitir relatórios.

6.1 Preencher os dados

No primeiro acesso a página restrita, realize o preenchimento dos seus dados para identificarmos se você é um contratante pessoa física ou jurídica. Clique no menu Perfil para cadastrar as suas informações (ver imagem 11), preencha todos os campos do formulário e ao final clique no botão Salvar.

Imagem 11 - Formulário da pessoa física (esquerda) e da pessoa jurídica (direita).

      Para alterar a senha, clique no nome do seu login localizado ao lado do menu Home. Na página que você será redirecionado, clique em Alterar Senha. Na página de Alterar senha preencha os campos Senha e Confirmar Senha. Clique no botão Alterar Senha para confirmar a alteração (ver imagem 6).

6.2 Ofertar vaga de emprego

Para ofertar vaga de emprego, clique no item Ofertar vaga localizado na parte inferior esquerda da sua página inicial (ver imagem 10). Na página Nova Vaga, preencha todos os campos do formulário e para finalizar o cadastro da vaga, clique no botão Criar Vaga (ver imagem 12).

Imagem 12 - Página do cadastro da vaga

6.3 Criar formulário de questões

        Após cadastrar a vaga de emprego, crie um formulário de questões para avaliar os seus candidatos selecionados. Na imagem 10, estão listadas algumas vagas cadastradas. O 1º passo é clicar no item Questionário referente a vaga que deseja criá-lo. Você será redirecionado para página Lista Questionários que contém a lista dos Questionários cadastrados para aquela vaga. Clique no item Novo Questionário (ver imagem 13) para cadastrar um novo questionário.

Imagem 13 - Página Lista Questionários

O 2º passo é definir o nome para o seu questionário no campo Descrição e clique no item Adicionar Questão (ver imagem 14). Para adicionar mais questões, basta clicar novamente em Adicionar Questão.

Imagem 14 - Página para criar um questionário.

Após clicar em Adicionar Questão, descreva a pergunta no campo Pergunta e adicione uma nota para esta questão (ver imagem 15). Em seguida, clique em Adicionar Opção para adicionar as respostas para a questão. Ao lado de cada resposta criada existe uma caixa que se for marcada indica que aquela resposta é a resposta correta (ver imagem 16).

Imagem 15  - Adicionar questões

Imagem 16  - Marcar a resposta correta

Ao finalizar a inclusão das perguntas e respostas para o questionário, clique no botão Salvar.

6.4 Selecionar candidatos

Para selecionar candidatos para a sua vaga clique no item Selecionar (ver imagem 10) referente a vaga. Na página de seleção de candidatos (ver imagem 17), será listado todos os candidatos que se candidataram aquela vaga. No lado direto do nome do candidato, clique no item selecionar para escolher o candidato. A palavra selecionado significa que o candidato já foi selecionado.

Imagem 17 - Página para selecionar candidato

6.5 Emitir relatórios

        O SELECAD+ gera as notas obtidas pelos candidatos após a resolução do questionário aplicado pelo contratante. Para ter acesso as notas siga os passos:

  • 1º clique no item Questionário da vaga desejada (ver imagem 10);
  • 2º clique no item Notas Candidatos do questionário desejado (ver imagem 18).

Após clicar em Notas Candidatos, o contratante será redirecionado para a página de Relatório de Notas por Candidato. Também é possível visualizar o currículo de cada candidato.

Imagem 18 - Relatório de Notas dos Candidatos
